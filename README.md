## рџ§  Solution Format

Each file follows this structure:

```python
# Problem: Two Sum
# Difficulty: Easy
# Link: https://leetcode.com/problems/two-sum/

# ----------------------------
# Approach: Hash Map
# Time Complexity: O(n)
# Space Complexity: O(n)
# ----------------------------

def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        diff = target - num
        if diff in seen:
            return [seen[diff], i]
        seen[num] = i
```
